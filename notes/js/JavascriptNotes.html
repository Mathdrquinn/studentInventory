Coding In Javascript

with Andrew Tomic and Brendan Quinn

What is a Variable

You can place data into these containers and then refer to the data simply by naming the container. Before you use a variable in a JavaScript program, you must declare it.

Variables are declared with the var keyword as follows: 

var foo = 3;

Now, if you were to call the variable you created called foo,
3 would be returned.

Functions

Functions are one of the fundamental building blocks in JavaScript. A function is a set of statements that performs a task or calculates a value.

To use a function, you must define it somewhere in the scope from which you wish to call it. 

function multiply(a, b) {
 return a * b; 
}

Now, if you were to call multiply(5, 10)
It would return 50.

Passing Data Into Functions

As shown above with functions, you can pass data into a function to perform tasks for you.

So, let's combine variables with functions like this: 

var thing1 = 3;
var thing2 = 6;

Now, let's call multiply that we created above and pass in our variables this and that: 

multiply(thing1, thing2);
// This will return 18.
