Creating an HTML File

  1. To create a file, right click on the desired directory in the left-hand view and click "New File". Name the file index.html
  2. Every HTML begins with the same basic structure 
    <!DOCTYPE html>
<html lang="en">
  <head></head>
  <body></body>
</html>
    • <!DOCTYPE html> tells the browser this is an HTML file
    • <html lang="en"></html> signals the start of the html
    • <head></head> will contain information about the page (Title, and other needed files)
    • <body></body> contains visual content for the page
  3. HTML files will be completely composed of elements
  4. Elements have one of three different default sizes based off of their display property.
    • inline (<a></a>, and <span></span>)
    • inline-block (<img />)
    • block (<main></main>, <ul></ul>, <li></li>, and <p></p>)
