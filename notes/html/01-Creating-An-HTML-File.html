Creating an HTML File

  1. To create a file, right click on the desired directory in the left-hand view and click "New File". Name the file index.html
  2. Every HTML begins with the same basic structure 
    <!DOCTYPE html>
<html lang="en">
  <head></head>
  <body></body>
</html>
    • <!DOCTYPE html> tells the browser this is an HTML file
    • <html lang="en"></html> signals the start of the html
    • <head></head> will contain information about the page (Title, and other needed files)
    • <body></body> contains visual content for the page
  3. HTML files will be completely composed of elements which are built by tags, attributes, and content
  4. Within the <head></head> element, there is always a <title></title> which gives the page its tab title. This is done by adding content to the element: e.g. <title>Irate Tangerines</title> yields
  5. Elements have one of three different default sizes determined by the display property.
    • inline
      • the element is no bigger than the size of its contents
      • e.g <a></a>, and <strong></strong>
    • inline-block
      • like inline, but it's size can be manipulated
      • e.g. <img />
    • block
      • the element grows to the size of its container
      • e.g. <main></main>, <ul></ul>, <li></li>, and <p></p>
