Introduction

This is a course designed to help you learn how to create your first web application.

To do that, we will be building a Student Inventory app. In doing so you will learn:

  • Basic HTML
  • How to add Styles to HTML
  • Basics and purpose of CSS
  • Beginner's Javascript
  • How to manipulate the DOM (HTML) using javascript

The course will be lead by instructors, but will have in it much built in work time. While this is the finished product, we will be asking you to recreate the application. Notes on the each phase can be found in the directory where they can be revisited at your leisure.
